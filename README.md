# automatiq-movies
A movie search app.

## Usage

1. Install
```
npm install
```

2. Add OMDB API Key by creating a `.env.local` file in the project root and add the following line:
```
OMDB_API_KEY={yourPrivateKey]
```

3. Run in local browser
```
npm run dev
```

4. Visit http://localhost:3000/automatiq/moviesSearch
